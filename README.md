## Featured Projects

### [HR Self-Service Information Desk](https://github.com/ashikdip/hr-desk)

An AI-powered HR helpdesk for organisations operating in **Finland** and **Bangladesh**. Built as a React single-page application with a 3-step login flow, bilingual UI (English/Finnish), and full Anthropic API integration.

### [Nagad Ltd. Recruitment Process Efficiency Analytics](https://github.com/ashikdip/nagad-recruitment-analytics)

End-to-end HR analytics on the recruitment pipeline of Nagad Ltd., Bangladesh's fastest-growing digital financial service (700+ employees). Covers funnel efficiency, time-to-hire bottlenecks, source channel ROI, and candidate dropout across 10,269 applicants and 130 requisitions.

> **Key finding:** Onboarding averages 18+ days - the single biggest bottleneck. Internal referrals produce 2x the hire rate of LinkedIn, yet remain an informal channel with no structured programme.

**Stack:** Python · Pandas · SQLite · Matplotlib · Seaborn · Jupyter

**Live dashboards:**
- [Recruitment Analytics Dashboard](https://ashikdip.github.io/nagad-recruitment-analytics/nagad-recruitment-dashboard.html) - 7 interactive charts with Nagad brand theming
- [2026 Forecast & Capacity Dashboard](https://ashikdip.github.io/nagad-recruitment-analytics/nagad-forecast-2026.html) - Linear regression forecast, TA capacity gap, strategic roadmap
